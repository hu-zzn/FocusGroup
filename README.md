# Focus_Group

Tech Stack

Frontend: Next.js

Backend: Node.js

Database & Auth: Supabase

Authentication: Role-based access control (RBAC) via Supabase Auth

Deployment: Vercel and TBD(Render or aws) 

Features
Role-Based Access Control (RBAC)

1) Admin

Manage users and roles

Monitor all cases and sessions

View analytics and system-wide data


2) Presenter

Create cases/topics

Add questions (MCQ, scale, text)

Launch focus group sessions

View responses and insights


3) Participant

Join sessions via link or code

View case details

Submit responses

Optional anonymous participation
